# Yaji Master E-commerce Website

A modern, mobile-first e-commerce website for Yaji Master, a premium Nigerian spice brand.

## Features

- **Mobile-First Design**: Fully responsive across all device sizes
- **WhatsApp Integration**: Direct ordering via WhatsApp
- **Shopping Cart**: Simple cart functionality with localStorage persistence
- **Premium Aesthetic**: Warm, spice-inspired color palette and clean typography
- **Product Catalog**: Showcase of 4 signature spice blends with 100g/200g options
- **Brand Storytelling**: Dedicated pages for brand story, mission, and recipes
